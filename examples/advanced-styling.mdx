## Grid Overlay

Show a grid overlay to visualize the character cell boundaries:

```svelte
<script lang="ts">
  import { AsciiArt } from 'svelte-asciiart';
  const text = `+----------+
|  Hello   |
|  World!  |
+----------+`;
</script>

<AsciiArt
  {text}
  grid
  gridClass="ascii-grid"
/>

<style>
  :global(.ascii-grid) {
    stroke: #87CEFA;
    stroke-width: 0.03;
    opacity: 0.5;
  }
</style>
```

<Tip>
The grid is useful for debugging alignment issues or creating a blueprint-style aesthetic.
</Tip>

## Frame Border

Add a border frame around your ASCII art:

```svelte
<script lang="ts">
  import { AsciiArt } from 'svelte-asciiart';
  const text = `+----------+
|  Hello   |
|  World!  |
+----------+`;
</script>

<AsciiArt
  {text}
  frame
  margin={1}
  frameClass="ascii-frame"
/>

<style>
  :global(.ascii-frame) {
    stroke: #FFB366;
    stroke-width: 0.05;
  }
</style>
```

<Note>
The frame respects the `margin` property, so increase margin to add space between the ASCII art and the frame.
</Note>

## Custom Fonts

Use custom monospace fonts from Google Fonts or other sources:

```svelte
<svelte:head>
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link
    rel="stylesheet"
    href="https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;600&display=swap"
  />
</svelte:head>

<script lang="ts">
  import { AsciiArt } from 'svelte-asciiart';
  const text = `+----------+
|  Hello   |
|  World!  |
+----------+`;
</script>

<AsciiArt
  {text}
  class="ascii-font-jetbrains"
/>

<style>
  :global(svg.ascii-font-jetbrains text),
  :global(svg.ascii-font-jetbrains tspan) {
    font-family: "JetBrains Mono", ui-monospace, monospace;
  }
</style>
```

<Note>
Popular monospace fonts include:
- JetBrains Mono
- Fira Code
- Source Code Pro
- IBM Plex Mono
- Courier New
- Consolas
</Note>


```svelte Stroke Effect
<script lang="ts">
  import { AsciiArt } from 'svelte-asciiart';
  const text = `+----------+
|  Hello   |
|  World!  |
+----------+`;
</script>

<AsciiArt
  {text}
  class="ascii-paint"
/>

<style>
  :global(svg.ascii-paint text),
  :global(svg.ascii-paint tspan) {
    fill: #ffffff;
    stroke: #000000;
    stroke-width: 0.5;
    paint-order: stroke fill;
  }
</style>
```

```svelte Both Fill & Stroke
<script lang="ts">
  import { AsciiArt } from 'svelte-asciiart';
  const text = `+----------+
|  Hello   |
|  World!  |
+----------+`;
</script>

<AsciiArt
  {text}
  class="ascii-paint"
/>

<style>
  :global(svg.ascii-paint text),
```

<Tip>
Use `paint-order: stroke fill` to ensure the stroke is drawn behind the fill, creating a clean outline effect.
</Tip>

## Complete Styled Example

Combine multiple styling techniques:

```svelte
<svelte:head>
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link
    rel="stylesheet"
    href="https://fonts.googleapis.com/css2?family=Fira+Code:wght@400;600&display=swap"
  />
</svelte:head>

<script lang="ts">
  import { AsciiArt } from 'svelte-asciiart';
  const text = `+----------+       _o<
|  Hello   |      \`\\,_
|  World!  |     (_)/ (_)
+----------+`;
</script>

<AsciiArt
  {text}
  rows={4}
  cols={22}
  cellAspect={0.6}
  grid
  frame
  margin={[1, 2, 1, 2]}
  class="ascii-font-fira ascii-bold ascii-bg ascii-paint"
  gridClass="ascii-grid"
  frameClass="ascii-frame"
/>

<style>
  :global(svg.ascii-font-fira text),
  :global(svg.ascii-font-fira tspan) {
    font-family: "Fira Code", ui-monospace, monospace;
  }

  :global(.ascii-bold) {
    font-weight: 700;
  }

  :global(.ascii-bg) {
    background: #f3f4f6;
  }

  :global(svg.ascii-paint text),
  :global(svg.ascii-paint tspan) {
    fill: #111827;
    stroke: #111827;
    stroke-width: 0;
    paint-order: stroke fill;
  }

  :global(.ascii-grid) {
    stroke: #87CEFA;
    stroke-width: 0.03;
    opacity: 0.5;
  }

  :global(.ascii-frame) {
    stroke: #FFB366;
    stroke-width: 0.05;
  }
</style>
```

<Note>
This example demonstrates:
- Custom font (Fira Code)
- Bold text
- Background color
- Custom text colors
- Grid overlay
- Frame border
- Custom margins
- Explicit dimensions
</Note>
