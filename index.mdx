<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-bold dark:text-white text-gray-900 mb-4">
    Quick start
  </h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with Svelte ASCII Art in minutes
  </p>
  
  <Steps>
    <Step title="Install the package">
      Install svelte-asciiart using your preferred package manager:

      <CodeGroup>
      ```bash npm
      npm install svelte-asciiart
      ```
      ```bash yarn
      yarn add svelte-asciiart
      ```
      ```bash pnpm
      pnpm add svelte-asciiart
      ```
      ```bash bun
      bun add svelte-asciiart
      ```
      </CodeGroup>
    </Step>

    <Step title="Import and use the component">
      Import the `AsciiArt` component and pass your ASCII art text:

      ```svelte
      <script>
        import { AsciiArt } from 'svelte-asciiart';

        const text = `+----------+
      |  Hello   |
      |  World!  |
      +----------+`;
      </script>

      <AsciiArt {text} />
      ```
    </Step>

    <Step title="Customize with props">
      Add grid overlays, frames, and margins to enhance your ASCII art:

      ```svelte
      <AsciiArt
        {text}
        grid
        frame
        margin={[1, 2]}
        gridClass="ascii-grid"
        frameClass="ascii-frame"
      />

      <style>
        :global(.ascii-grid) {
          stroke: #90ee90;
          stroke-width: 0.03;
          opacity: 0.5;
        }
        :global(.ascii-frame) {
          stroke: #ffb366;
          stroke-width: 0.05;
        }
      </style>
      ```
    </Step>
  </Steps>
</div>
